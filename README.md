# moestation
 WIP PlayStation 2 emulator written in Zig. Very incomplete and *extremely* slow.

# Compatibility
 - Atelier Iris: Eternal Mana gets to the title screen.

# Dependencies
 - SDL2
 - SDL.zig (https://github.com/MasterQ32/SDL.zig)
